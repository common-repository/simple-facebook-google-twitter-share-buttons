This plugin adds facebook, google, twitter share button on your WordPress site

== Description ==

Simple Facebook Google Twitter Share Buttons (SFGTSB) is a lightweight plugin that adds facebook, google, twitter share button, simple and swift.

Place 'sfgtsb_share_button();' in your templates.

== Installation ==

1. Upload `simple-facebook-google-twitter-share buttons` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Place 'sfgtsb_share_button();' in your templates.
